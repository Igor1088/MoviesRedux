## Includes

* react
* react-router
* redux
* redux-thunk
* Bootstrap
* TMDb API

## Run on your Machine

1. git clone `https://github.com/Igor1088/MoviesRedux.git`
2. cd MoviesRedux 
3. `npm install`
4. `npm start`
